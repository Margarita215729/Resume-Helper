## 🚀 АКТИВАЦИЯ GITHUB PAGES - 3 ПРОСТЫХ ШАГА

### ✅ ШАГ 1: Включить GitHub Pages
1. Откройте: https://github.com/Margarita215729/Resume-Helper/settings/pages
2. В разделе "Source" выберите **"GitHub Actions"**
3. Нажмите **Save**

### ✅ ШАГ 2: Создать GitHub Token для AI
1. Откройте: https://github.com/settings/tokens
2. Нажмите **"Generate new token (classic)"**
3. Имя: `Resume Helper AI`
4. Срок: **90 дней**
5. Разрешения: ✅ **repo**, ✅ **workflow**
6. Нажмите **Generate** и скопируйте токен

### ✅ ШАГ 3: Добавить токен в секреты
1. Откройте: https://github.com/Margarita215729/Resume-Helper/settings/secrets/actions
2. Нажмите **"New repository secret"**
3. Имя: `GITHUB_TOKEN`
4. Значение: вставьте ваш токен
5. Нажмите **Add secret**

### 🎯 ЗАПУСК РАЗВЕРТЫВАНИЯ
- Зайдите: https://github.com/Margarita215729/Resume-Helper/actions
- Выберите **"Deploy to GitHub Pages"**
- Нажмите **"Run workflow"** → **"Run workflow"**
